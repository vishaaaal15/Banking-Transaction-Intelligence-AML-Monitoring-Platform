# Banking Transaction Intelligence & AML Monitoring Platform

## Overview

The Banking Transaction Intelligence & AML Monitoring Platform is an end-to-end banking analytics solution designed to monitor transaction activity, identify suspicious patterns, assess customer risk exposure, and support Anti-Money Laundering (AML) investigations through data-driven analytics.

This project integrates SQL, Python, Excel, and Power BI to analyze transaction behavior, detect anomalies, monitor compliance KPIs, and provide executive-level insights through interactive dashboards.

---

## Business Problem

Financial institutions process thousands of transactions daily, making it challenging to identify suspicious activities, unusual transaction patterns, and potential compliance risks.

The objective of this project is to:

- Monitor transaction activity across banking channels
- Identify suspicious transaction patterns
- Evaluate customer risk exposure
- Support AML monitoring and compliance reporting
- Detect unusual transaction behaviors
- Enable proactive fraud and financial crime investigations

---

## Project Objectives

- Analyze banking transaction behavior
- Monitor AML and compliance KPIs
- Identify high-risk customers and transactions

## Dataset Overview

The dataset consists of simulated banking transaction data including:

- Customer Information
- Transaction Records
- Transaction Amounts
- Payment Channels
- Account Types
- Transaction Categories
- Customer Risk Scores
- Geographic Information
- AML Risk Indicators
- Suspicious Activity Flags

### Dataset Scale

- 500,000+ Banking Transactions
- ₹1.2B+ Transaction Volume Evaluated
- Multi-Channel Banking Activity
- Multi-Region Customer Coverage
